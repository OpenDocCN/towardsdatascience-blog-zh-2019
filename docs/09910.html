基尼杂质:

基尼系数是以意大利统计学家科拉多基尼命名的。基尼不纯可以理解为一个最小化误分类概率的标准。为了理解决策树的定义(如图所示)以及我们如何构建决策树，让我们从一个非常简单的数据集开始，根据不同的天气条件，我们决定是否玩户外游戏。根据定义，仅包含一个类别的数据集将具有 0 基尼系数杂质。在建立决策树时，我们的想法是选择基尼系数最小的特征作为根节点，以此类推...让我们从简单的数据集开始——
