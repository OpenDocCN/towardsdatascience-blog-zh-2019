# **应该学什么？**

想象你又是一个孩子，你不知道该选哪个专业。

在美国，科学、技术、工程和数学(STEM)专业的平均薪酬往往比商业和人文科学、社会科学或生命科学专业的同龄人高出 20%。与此同时，在欧洲，分布完全相反，STEM 比他们的 HSSLS 同行少挣大约 20%。

![](img/72decde0ddd224ed3520a7cb48443677.png)

一旦你选择了一个专业，你最终会面临选择是继续深造还是加入劳动大军。
