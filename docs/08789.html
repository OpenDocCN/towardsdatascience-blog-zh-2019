音频信号的频谱图特征

它是信号频谱随时间变化的直观表示。可以使用几种方法生成频谱图，包括傅立叶变换、小波变换和带通滤波器。

设 x 是长度为 n 的信号。考虑长度为 m 的 x 的连续段(或“片段”)，其中 m ≤ n，设 X ∈ R ^(m×(N−m+1)是以连续段作为连续列的矩阵。换句话说，[x[0]，x[1]，。。。，x[m 1]]^t 是第一列，[x[1]，x[2]，。。。x[m]]^T 是第二列，依此类推。X 的行和列都是按时间索引的。
